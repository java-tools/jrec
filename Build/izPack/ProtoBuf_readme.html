ProtoBuf Editor

This program is for editing Binary Files in Googles Protocol Buffers format. It is built on top of the RecordEditor Project (RecordEditor 0.92) and uses Protocol Buffer 2.5.0

The editor requires Protocol buffers 2.5.0 (or later ???) to be installed. The Protocol Buffer programs need to be on the Program Path, it uses the protoc exe to compile proto files.

For later versions of Protocol Buffers, it may be possible to replace the ProtoBuffers.jar with a later protocol buffers jar (this will work if Google maintains backward compatibility) ???
