Sort Panel

You can start the Sort option from either the Right Click Menu and from the Data Drop down Menu. On this screen you specify which fields to sort on.

The Screen consists of

  • Record Selection at the top, Typically you would only Sort when there is only one record Type present.
  • Sort Fields fields to sort on and wether to sort in a Ascending / Descending Direction.
  • Use Wether to Sort the whole file or just the Selected records.
  • Sort button - used to start the Sort process.
