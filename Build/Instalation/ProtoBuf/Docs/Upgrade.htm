Upgrading the DB

All the programs and supporting files should of been updated by the install, you just need to update the database. To do this

  1. Start the DB server (not needed for the MS Access version).
  2. Start the Record Layout Edit program.
  3. Take the Upgrade Database Tables option at the bottom of the menu.
  4. Run the appropriate upgrade steps. For example if you are currently running version 0.60 you need to
    • run Upgrade tables from 0.55 - 0.61b to 0.62

You should then be able to run the RecordEditor.
