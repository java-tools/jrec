Record Editor

Some important points before you start using the RecordEditor:

  1. There should be a new option RecordEditor_Generic on your Start Menu where all the record edit options / Documentation is available.
  2. The Full Editor is the main data file editor. It can be started from the RecordEditor_Generic option or via bat / shell scripts in the $install_Directory/lib directory.
  3. The Layout Editor is used to edit Record layouts and to import Cobol Copybooks. It can be started from the RecordEditor_Generic option or via bat / shell scripts in the $install_Directory/lib directory.
  4. The Record Editor is cut down version of the full editor, and is used to edit Data Files. It can be started from the RecordEditor_Generic option or via bat / shell scripts in the $install_Directory/lib directory.
  5. For Windows User's, there is a Registry File (lib\HSQL.reg) to make the RecordEditor available on the Right Click in Windows Explorer. You may need to edit this file.
