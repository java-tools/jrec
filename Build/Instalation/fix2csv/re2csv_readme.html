Instalation

This program requires that the RecordEditor be installed. You should unzip it to where ever the RecordEditor package is installed. This package is primary an example of how to use the RecordEdit / JRecord code

Compiling (Eclipse etc)

The source code references classes in the RecordEdit.Jar which is part of the RecordEditor package install. This jar must be included in the build path.

Using the package

There are 2 basic wys to use this package

  • Online - use re2CsvGui.bat
  • Batch - use re2Csv.bat

    Usage: re2Csv.bat -l layout-name -i input-file -o output-file -s field-seperator


    There are 2 examples provided re2csvtest1.bat and re2csvtest2.bat
