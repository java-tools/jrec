Cobol Editor

  • Further Details

    • Cobol Editor

    When the Record Editor starts, it displays the Edit Start Screen 

    Fields on this screen are

    

    Field

    Description

    File

    Name of the file to edit, you can select a file by using the Find File button to the
right.

    Copybook Type

    Type of copybook (record layout). Currently the choices are
Cobol and cb2xml XML, but it possible to write your own
copybook reader.

    Split Copybook

    Wether to split the copybook into seperate records (on either 01 level or redefines).

    Copybook

    Copybook File name. You can use the Choose File to search the
directory for the copybook file.

    File Structure

    File Structure (or file reader to use). For Windows/Unix/Linux text files
use either Default Reader or Text IO.

    Numeric Format

    Format of numeric data.

    Field Seperator
    Field Seperator to use in a CSV file

    Quote
    Quote to use in CSV files

    

    

    Buttons on the Screen

    

    Button

    Description

    Choose File

    Start the File Chooser dialogue which will navigate the File system and the user to choose a file.

    Choose Layout

    Choose Record Layout file (ie Cobol Copybook).

    Edit

    Edit the file.

    Browse

    View the file, but no changes are possible.
