Sort Tree

Sort Tree view is a way to summarise the Data in a File. The program will sort the file and group records with the same sort Keys together to form a Tree. Longer Term I hope to display a Summary Record for each Tree Node.

Starting a Sort Tree view

First select the Sorted Tree View option of the View Menu.

The Screen consists of

  • Record Selection at the top, Typically you would only use a Sort Tree when there is only one record Type present.
  • Sort Fields fields to sort then build the Tree on.
  • Summary Table supply the summary function for the fields.
  • Use Wether to build the Tree from the whole file or just the Selected records.
  • Build button - used to start the build process.
  • Save Button Saves the sort tree definition to an external file. You can run it later via the view menu, see for more details.
