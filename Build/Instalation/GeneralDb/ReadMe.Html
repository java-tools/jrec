Requirements

The source code of RecordEditor has the following requirements (all optional):

  • cb2xml and Table Layout. Both these 2 packages are included in the download in the lib directory of the zip file. ie lib/cb2xml.Jar and lib/TableLayout.jar
  • Test Classes use JUnit. If you do not have JUnit, these classes can be deleted (they are only used for to automate the testing).
  • Example - Some of the examples use Velocity. Again these classes can be deleted if required.
