Starting the Record Editor

The Record Editor can either be started from

  • The start Menu
  • From Windows Explorer, you can use the right click button -

  • From the command line. It accepts the following parameters
    1. File Name
    2. Line Number (preceded by -l).

    To edit a file Filename and move to the 100 record you would enter

    java.jar -jar .../RecordEdit.jar Filename -l 100

When the Record Editor start it displays the record Layout Selection Screen 

Fields on this screen are



Field

Description

File

File Name, you can select a file by using the Find File to the
right.

Database

The system stores it record definition in a database. The system can also
connect to multiple Databases. This allows you have different Test / Production
Layout DB's. This Combo is used to choose which DB to use.

System

The Record Layouts can be grouped into different systems. This Combo Box is
used to select a specific System.

Record Layout

The Record Layout used to display the file. Only layouts belonging to the
selected system (above) are displayed.





Buttons on the Screen



Button

Description

Find File

Used to select the file

Reload

Reload System Details from the DB.

Edit

Edit the file

Browse

View the file, but no changes are possible
