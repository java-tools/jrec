Compiling the Source

Its assumed that you have RecordEditor installed, you must copy velocity-1.7.jar andvelocity-dep-1.7.jar from your RecordEditor lib to the lib directory in this zip file !!!!!!

In the Source Directory, there are the following sub directories.

DirectoryContent
cb2xmlholds my version of cb2xml - Cobol Copybook analysis
JRecord_CommonLow level IO and line representation code that is shared with the JRecord
JRecordMid level IO and line representation code
RecordEditorSource for the editor
RecordEditor_TestsRecord Editor Tests
re_utilitiesRecordEditor utilities holds JUnit tests + some example code.
ZCalendarDate Display jar + lowlevel ComboPopup code.
