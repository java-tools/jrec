Record Editor

Some important points before you start using the RecordEditor:

  1. There should be a new option RecordEditor_HSQLDB on your Start Menu where all the record edit options / Documentation is available.
  2. You must start the HSQL Database server before you can run the record editor or layout editor. There should be an option to start the Database Sever option under RecordEditor_HSQLDB on your Start Menu.
    Otherwise there is $install_Directory/lib/StartServer.bat or $install_Directory/lib/StartServer.sh.
  3. The Full Editor is the main data file editor. It can be started from the RecordEditor_Generic option or via bat / shell scripts in the $install_Directory/lib directory.
  4. The Layout Editor is used to edit Record layouts and to import Cobol Copybooks. It can be started from the RecordEditor_HSQLDB option or via bat / shell scripts in the $install_Directory/lib directory.
  5. The Record Editor is a cut down version of Full Editor. Ir is also used to edit Data Files. It can be started from the RecordEditor_HSQLDB option or via bat / shell scripts in the $install_Directory/lib directory.
  6. For Windows User's, there is a Registry File (lib\HSQL.reg) to make the RecordEditor available on the Right Click in Windows Explorer. You may need to edit this file.
