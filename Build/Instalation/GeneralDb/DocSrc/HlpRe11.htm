Field Tree

A field Tree is like the Sort Tree (except there is no sort). It will group records with the same field values together as a Tree.

The Screen consists of

  • Record Selection at the top, Typically you would only use a field Tree when there is only one record Type present.
  • Field fields to group by.
  • Use Wether to build the Tree on the whole file or just the Selected records.
  • Build Tree button - used to start the Build Tree process.
